# Contributing to Excel Template Filler

Thanks for stopping by! Built by an engineer in the field, for engineers in the field — contributions that make batch template filling more robust are always welcome.

## Ways to Contribute

- **Report bugs** — Open an [Issue](../../issues) with your template type + error output
- **Suggest features** — Start a [Discussion](../../discussions) to talk through ideas
- **Submit PRs** — See below

## Pull Request Process

1. Fork the repo and create a branch (`feat/description` or `fix/description`)
2. Keep changes focused — one feature or fix per PR
3. Test with a real `.xlsx` template (not just synthetic data)
4. Open a PR against `main`

## Code Style

- Python 3.10+
- Type hints where practical
- Comments in English
- Docstrings for public methods

## Questions?

Start a [Discussion](../../discussions) — no question is too small.
